# 🌍 WeatherMeter — Live Global Weather Dashboard

A **Worldometer-style** live global weather dashboard built with **Next.js**, **Open-Meteo API**, **Leaflet**, and animated counters.

## Features

- 🌡️ Live weather updates from Open-Meteo (no API key required)
- 🗺️ Interactive world map with city markers
- 🔢 Animated Worldometer-style counters
- ⚡ Server-side caching (5-minute intervals)
- 🚀 Deployable on Vercel in minutes

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Tech Stack

- **Next.js 14** — React framework with API routes
- **Open-Meteo** — Free weather API (no key needed)
- **Leaflet + react-leaflet** — Interactive maps
- **Axios** — HTTP client

## Deploy

1. Push to GitHub
2. Import on [Vercel](https://vercel.com)
3. Deploy — no environment variables needed!
